The Grid Language (work in progress)

James Geddes

The following material is work in progress

Grid is a language for describing spreadsheets, although to call it a “language” is perhaps overblown. It is declarative, in the sense that it merely describes the structure and content of a spreadsheet. A grid program may be straightforwardly converted into a specific spreadsheet application, such as the Open Document Format for Office Applications (ODF).

We don’t define an external form for a grid program. Instead, it exists as an internal data structure (perhaps generated by a cell program). The following is a semi-formal description.
